Sobre mim

Olá, me chamo João Victor, sou um entusiasta em programação, atualmente estou focando meus estudos em desenvolvimento web. Meu trabalho é projetar e desenvolver a interface do usuário de um site ou aplicativo. As linguagens utilizadas para desenvolver aplicações web incluem HTML, CSS e JavaScript. O HTML é a linguagem de marcação utilizada para estruturar o conteúdo da página web. O CSS é utilizado para estilizar a página, incluindo as cores, fontes e layouts. Já o JavaScript é utilizado para adicionar interatividade à página, permitindo que o usuário interaja com elementos da página.
